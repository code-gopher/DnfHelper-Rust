## 🔧 技术亮点

### 1. CALL 注入技术
```rust
// 自动生成汇编代码
asm_code.extend_from_slice(&[0x60]); // pushad
asm_code.extend_from_slice(&[0x9C]); // pushfd
// 压入参数
for &param in params.iter().rev() {
    asm_code.extend_from_slice(&[0x68]);
    asm_code.extend_from_slice(&(param as u32).to_le_bytes());
}
// CALL 目标
asm_code.extend_from_slice(&[0xE8]);
// 恢复现场并返回
```

### 2. 线程安全设计
- 使用 `Arc<RwLock<T>>` 实现全局状态
- `AtomicBool` 控制线程启停
- 无数据竞争

### 3. 跨平台兼容
- Windows: 完整功能
- Linux/Mac: 编译通过，功能占位

## ⚠️ 注意事项

1. **地址需要实际调试**: 代码中的内存地址是示例值，实际使用需要通过 CE 等工具调试获取
2. **需要管理员权限**: Windows 下操作其他进程需要管理员权限
3. **反作弊风险**: 实际游戏可能有反作弊系统，仅供学习研究
4. **版本依赖**: 不同游戏版本内存地址不同
